# Cover Letter Generator

This Raycast extension helps you create professional cover letters for job applications instantly using the Gemini API. It analyzes the job description from your selected text and your uploaded resume to generate a tailored cover letter.

## Features

- **Instant Generation:** Create a cover letter in seconds based on a job description.
- **Resume Analysis:** Uses your actual resume as context for the letter.
- **History Tracking:** Keep track of all generated cover letters, including company names and match scores.
- **PDF Export:** Export your generated cover letters directly to PDF.

## Getting Started

### 1. Prerequisites

To use this extension, you'll need a Google Gemini API Key.

### 2. Obtain a Gemini API Key

1. Go to the [Google AI Studio](https://aistudio.google.com/app/apikey).
2. Create a new API key or use an existing one.
3. Copy the API key.

### 3. Setup

1. Open Raycast and find the **Cover Letter Generator** extension.
2. Enter your **Gemini API Key** in the extension preferences.
3. You can also select your preferred Gemini model (e.g., Gemini 1.5 Flash or Pro) in the settings.

### 4. Usage

1. Select the job description text on any webpage or document.
2. Run the **Create Cover Letter** command.
3. Upload your resume (PDF) if you haven't already.
4. The extension will generate a cover letter, show you a match score, and extract key skills.
5. You can then edit the letter, copy it to your clipboard, or export it as a PDF.
